# stpw

## Steps:
### 1. Create and activate virtual environment and install dependencies using requirements.txt
### 2. set appropriate parameters in Makefile
### 3. Generate c files:
``` bash
make c
```
### 4. Generate assembly files
``` bash
make asm
```
### 5. Generate datasets
``` bash
make data
```
### 6. To delete all temporary files
``` bash
make clean
```
